# API de Produtos – Node.js e Express

Esta é uma **API REST simples** desenvolvida com **Node.js** e **Express**, criada com fins de **aprendizado** para praticar conceitos básicos de backend, rotas HTTP e manipulação de dados em memória.

A API permite **listar produtos** e **cadastrar novos produtos**, utilizando requisições **GET** e **POST**.
Os testes das rotas foram realizados com o **Postman**.

---

## Tecnologias utilizadas

* Node.js
* Express
* JavaScript
* Postman (para testes das requisições)
* npm (package.json)

---

## Funcionalidades

* Listar todos os produtos cadastrados
* Criar um novo produto
* Retorno de respostas em formato JSON
* Uso correto de códigos de status HTTP (200 e 201)
